# Крестики-нолики.

Поиграть можно [localhost:3000](http://localhost:3000/) 

## Для игры запустите backend:
```
cd backend && node app
```
## Затем запустите frontend
```
npm start
```


## Установка бэкенда
```
сd ttt/backend/
npm i
node app
```

## Установка фронтенда
```
сd ttt
npm i
npm start
```
